== Présentation
Créer des DAO est une tâche répétitive et ingrate, 90% du code est
similaire d'un DAO à l'autre. Spring propose une méthode standard pour
gérer les DAO au travers d'objets, ou plutôt d'interfaces, de type
`CrudRepository`.
